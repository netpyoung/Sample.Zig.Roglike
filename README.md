# Sample.Zig.Roglike

This project is a small Roglike implementation for learning Zig while keeping up with changes across Zig versions.

- [zig](https://ziglang.org/)

``` cmd
> zig version
0.15.2
```

## Used

- Lib:
  - [libsdl-org/SDL](https://github.com/libsdl-org/SDL)
  - [libtcod/libtcod](https://github.com/libtcod/libtcod)
- IDE: [LuckystarStudio.ZigVS](https://marketplace.visualstudio.com/items?itemName=LuckystarStudio.ZigVS)
